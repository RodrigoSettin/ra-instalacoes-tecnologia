# 📊 Previsão de Demanda e Otimização de Estoque

---

## 📌 Descrição do Projeto

Projeto desenvolvido com foco na aplicação de análise de dados e Machine Learning para previsão de demanda, apoiando decisões estratégicas de estoque e operação.

A proposta demonstra como dados históricos podem ser utilizados para antecipar necessidades, reduzir desperdícios e melhorar a eficiência operacional.

---

## 🎯 Objetivo

- Analisar padrões de consumo
- Prever demanda futura
- Apoiar decisões de reposição de estoque
- Reduzir riscos de falta ou excesso de produtos

---

## ❓ Problema de Negócio

Em operações reais, é comum enfrentar:

- Falta de produtos em momentos críticos
- Excesso de estoque parado
- Decisões baseadas apenas em experiência

**Pergunta-chave:**  
Como prever a demanda de forma mais precisa utilizando dados?

---

## 🧠 Solução Aplicada

- Utilização de dados históricos de vendas
- Análise exploratória dos dados
- Construção de modelo preditivo utilizando **AWS Canvas (AutoML)**
- Geração de previsões para apoio à tomada de decisão

---

## 🛠️ Tecnologias Utilizadas

- AWS Canvas (Machine Learning - AutoML)
- Dados estruturados em CSV
- GitHub

---

## 📊 Dados Utilizados

O dataset contém informações como:

- Quantidade vendida por período
- Categoria do produto
- Tipo de mídia
- Eventos promocionais e lançamentos
- Comportamento de demanda

*Dados simulados para fins educacionais.*

---

## 📈 Resultados Esperados

- Melhor previsibilidade de demanda
- Redução de perdas e desperdícios
- Apoio estratégico à operação
- Aplicação prática de Machine Learning em problema de negócio

---
